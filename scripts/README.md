# Scripts (dev / test / diagnose)

These are for development and troubleshooting. They are not included in the zip installer.

- **install_service.bat**, **uninstall_service.bat**, **configure_service_account.bat** – install/uninstall service when running from source
- **run_standalone.bat**, **run_service_python.bat**, **run_service_terminal.bat**, **run_service_with_logging.bat** – run app or service from source
- **diagnose_service.bat**, **diagnose_crash.bat**, **diagnose_install.bat** – diagnostics
- **find_crash.bat**, **find_port_8080.bat** – helpers
- **test_*.bat**, **test_*.py** – tests
- **fix_service.bat**, **post_install.bat**, **check_windows_errors.bat** – misc
